// how it works

opclaude runs a small proxy on 127.0.0.1 that speaks Anthropic's API to Claude Code on one side, and your opencode Go subscription on the other. Claude Code talks to it exactly like it talks to Anthropic — same CLI, same flags — while opclaude smooths over each model's quirks behind the scenes. Your opencode key never leaves your machine.

Pulls the repo into ~/.opclaude-src and runs the installer — prompts to install uv and Claude Code if missing, then asks for your opencode API key. Needs an opencode Go subscription. MIT licensed.
